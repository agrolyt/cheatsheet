Common CSS properties

Properties Description
background Shorthand for background- properties
background-color Specifies background color of an element
border shorthand property for border-width, border-style and border-color
background-image Specifies one or more background images for an element
color Sets the color of text
text-align Specifies the horizontal alignment of text

CSS selectors and specificity

Selectors Example Description
element p selects all <p> elements
class .code selects all elements with class="code"
id #intro selects all elements with id="intro"
